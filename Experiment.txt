# ----------------------------------------------------------------------
# 1. Oddball トライアル
# ----------------------------------------------------------------------
# n = 10, ratio = 0.2 の練習用トライアル
# 課題の操作確認と音量調整を兼ねる
#
# 画面表示
# 「これから練習を行います．
#
#   低い音と高い音がランダムに流れます．
#   高い音が鳴ったときに，SPACEキーを押してください．
#
#   準備ができたらSPACEキーを押してください．」
#
# SPACEキー入力後，3秒のカウントダウン
#
# 「3」
# 「2」
# 「1」
# 「開始」
#
# Oddball練習開始
#
# 練習終了後
# 「練習は終了です．
#
#   音量や操作方法に問題がある場合は，
#   実験者に伝えてください．」
#
# 実験者が音量と操作を確認
#
# 「問題がなければ，SPACEキーを押してください．」
#
#
# ----------------------------------------------------------------------
# 2. 安静
# ----------------------------------------------------------------------
# 2分間の安静状態を計測
#
# 画面表示
# 「これから2分間，静かにお待ちください．
#
#   できるだけ身体を動かさず，
#   リラックスして過ごしてください．
#
#   終了時に音が鳴ります．
#
#   準備ができたらSPACEキーを押してください．」
#
# SPACEキー入力後，開始
#
# 画面中央に「＋」を表示
# 120秒間待機
#
# 120秒経過後，短い終了音を鳴らす
#
# 画面表示
# 「終了です．
#   続いてアンケートに回答してください．」
#
#
# ----------------------------------------------------------------------
# 3. アンケート
# ----------------------------------------------------------------------
# 安静状態についてアンケートに回答
#
# 画面表示
# 「アンケートに回答してください．
#
#   回答が完了したらSPACEキーを押してください．」
#
# Google Formをタブレット端末で回答
#
# アンケート回答中に，1回目のOddball課題をgenerateする
#
# 画面表示
# 「これから1回目の課題を行います．
#
#   高い音が鳴ったときに，
#   SPACEキーを押してください．
#
#   準備ができたらSPACEキーを押してください．」
#
#
# ----------------------------------------------------------------------
# 4. Oddball 課題 1
# ----------------------------------------------------------------------
# n = 200, ratio = 0.2 の本番課題
#
# SPACEキー入力後，3秒のカウントダウン
#
# 「3」
# 「2」
# 「1」
# 「開始」
#
# Oddball課題開始
#
# 課題中は画面中央に「＋」を表示
# 標準刺激およびターゲット刺激の提示時にTTL信号を送信
# 高い音に対するSPACEキー入力を記録
#
# 課題終了後
# 「1回目の課題は終了です．
#   続いてアンケートに回答してください．」

#
